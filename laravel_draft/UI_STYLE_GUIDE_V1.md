# UI_STYLE_GUIDE_V1

## Theme
- Light, neutral professional palette
- Restrained indigo/blue accents
- Soft borders, subtle shadows, compact radii

## Core Tokens
- Background: `#f5f7fb`
- Panel: `#ffffff`
- Border: `#e2e8f0`
- Text: `#0f172a`
- Muted: `#64748b`
- Primary: `#3347a9`
- Success: `#0f766e`
- Warning: `#b45309`

## Typography
- Primary stack: Inter/Segoe UI/Arial
- Page title: 22px, strong
- Section title: 16px
- Body UI: 13px
- Helper/meta: 12px muted

## Components
- Card: border + subtle shadow + 12px radius
- Buttons: primary/secondary/success/warn/danger variants
- Inputs/selects: consistent size, strong focus ring
- Tables: separate headers, hover rows, cleaner density
- Badges: semantic pills for state emphasis
- Empty states: dashed neutral placeholders
- Result consoles: dark monospace `pre` with rounded container

## Layout
- 12-column responsive grid
- page header with breadcrumb + subtitle
- sidebar grouped into module clusters
- compact operator flow, low scrolling cost
